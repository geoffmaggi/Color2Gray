# Color 2 Gray
  
  Based off Color2Gray: Salience-Preserving Color Removal. Amy A. Gooch, Sven C. Olsen, Jack Tumblin, and Bruce Gooch. SIGGRAPH 05
  http://www.cs.northwestern.edu/~ago820/color2gray/color2gray.pdf
